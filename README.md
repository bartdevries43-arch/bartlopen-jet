# bartlopen — Run Coach · Jet

Jets persoonlijke zomer-opbouwschema (hardlopen–wandelen) richting de 10 km van
Rotterdam (zondag 11 april 2027). 7 weken, 2 trainingen per week (ma + do),
voorzichtig opgebouwd met aandacht voor een gevoelige rechtervoet.

Statische web-app — werkt op telefoon, laptop en iPad, en is te "installeren" als app.
